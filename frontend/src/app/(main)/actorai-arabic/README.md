# ActorAI Arabic Studio

## Overview
The ActorAI Arabic Studio is a specialized platform for actors and performers to analyze and rehearse Arabic scripts using AI-powered tools. It provides features for script analysis, AI scene partnering, and performance analytics tailored specifically for Arabic language content.

## Directory Structure

### Components
- `ActorAiArabicStudio.tsx`: Main component that renders the studio interface
- `components/`: Contains all UI components used in the studio

### Static Source
- `static-source/`: Contains static assets and source files for the studio
  - `app.js`: Main JavaScript application file
  - `index.html`: Main HTML file for the studio
  - `style.css`: Main CSS stylesheet for the studio

### Pages
- `page.tsx`: Next.js page component that renders the ActorAI Arabic Studio

## Key Features
1. **Script Analysis**: Deep analysis of Arabic scripts using proven acting methodologies
2. **AI Scene Partner**: Virtual scene partner that responds naturally to your performance
3. **Performance Analytics**: Detailed feedback on emotional authenticity, vocal delivery, and physical presence
4. **Progress Tracking**: Monitor your growth with comprehensive analytics and personalized coaching tips

## Technical Implementation
The studio is built using a combination of React components and vanilla JavaScript for the static source files. It follows a component-based architecture with clear separation of concerns between UI components and business logic.

## Usage
To use the ActorAI Arabic Studio, navigate to the studio page and follow the on-screen instructions to upload your script, analyze it, and rehearse with the AI scene partner.
