# ActorAI Arabic Studio Components

## Overview
This directory contains all the React components used in the ActorAI Arabic Studio. These components provide the UI and interactive elements for the studio's features.

## Key Components

### ActorAiArabicStudio.tsx
- Main component that renders the studio interface
- Manages the overall state and navigation between different studio features
- Contains the main layout and structure of the studio

## Component Structure
The components follow a modular architecture with clear separation of concerns. Each component is responsible for a specific part of the UI and its associated functionality.

## Usage
These components are used internally by the ActorAI Arabic Studio and should not be used directly in other parts of the application. To use the studio, navigate to the studio page and follow the on-screen instructions.
