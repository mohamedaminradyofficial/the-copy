# المجلد: analysis_output

## وصف المجلد
يحتوي هذا المجلد على ملفات ناتجة من عملية تحليل النصوص الدرامية. هذه الملفات تحتوي على نتائج التحليل، مثل التقارير النهائية، التصورات المرئية، والتحليلات الإحصائية.

## قائمة بالملفات ووظائفها

1. **final-report.json**
   - وصف: يحتوي على تقرير تحليل شامل للنص الدرامي بتنسيق JSON.
   - محتوى: يتضمن ملخص تنفيذي، تحليل نقاط القوة والضعف، فرص التحسين، التهديدات للتماسك السردي، وتقييم عام للنص.
   - استخدام: يمكن استخدامه من قبل المطورين أو المحللين لقراءة النتائج بشكل آلي.

2. **final-report.txt**
   - وصف: يحتوي على تقرير تحليل شامل للنص الدرامي بتنسيق نصي.
   - محتوى: يتضمن ملخص تنفيذي، تحليل نقاط القوة والضعف، فرص التحسين، التهديدات للتماسك السردي، وتقييم عام للنص.
   - استخدام: يمكن استخدامه من قبل المطورين أو المحللين لقراءة النتائج بشكل نصي.

3. **visualizations.json**
   - وصف: يحتوي على تصورات مرئية وتحليلات إحصائية للنص الدرامي بتنسيق JSON.
   - محتوى: يتضمن رسوم الشبكة، تصورات زمنية، رسوم إحصائية، وعناصر تفاعلية.
   - استخدام: يمكن استخدامه من قبل المطورين أو المحللين لقراءة التصورات المرئية بشكل آلي.

4. **visualizations.txt**
   - وصف: يحتوي على ملخص للتصورات المرئية وتحليلات الإحصائيات للنص الدرامي بتنسيق نصي.
   - محتوى: يتضمن ملخص للرسوم الشبكة، التصورات الزمنية، الرسوم الإحصائية، وعناصر التفاعلية.
   - استخدام: يمكن استخدامه من قبل المطورين أو المحللين لقراءة التصورات المرئية بشكل نصي.

## ملاحظات خاصة
- جميع الملفات في هذا المجلد هي ملفات ناتجة من عملية تحليل تلقائية، ولا يجب تعديلها يدوياً.
- يمكن استخدام هذه الملفات من قبل المطورين أو المحللين لقراءة النتائج والتحليلات بشكل آلي أو نصي.
- لا توجد اعتمادات أو علاقات مع مجلدات أخرى في هذا المجلد.
