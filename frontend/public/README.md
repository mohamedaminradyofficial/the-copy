# المجلد العام

## وصف المجلد
يحتوي هذا المجلد على الملفات العامة التي يمكن الوصول إليها من قبل المتصفح. يتضمن هذا المجلد صورًا، خطوطًا، ورسوماتًا، وملفات CSS، وواجهات برمجة التطبيقات (APIs) العامة، وملفات JavaScript العامة، وملفات JSON العامة، وملفات XML العامة، وملفات SVG العامة، وملفات TTF العامة، وملفات WOFF العامة، وملفات WOFF2 العامة، وملفات EOT العامة، وملفات OTF العامة.

## قائمة الملفات ووظائفها
- `directors-studio/`: يحتوي على صورًا ورسوماتًا تستخدم في واجهة المستخدم.
- `fonts/`: يحتوي على خطوطًا تستخدم في واجهة المستخدم.
- `pdf-worker/`: يحتوي على ملفات JavaScript العامة التي تستخدم في عرض ملفات PDF.

## ملاحظات خاصة
- يتم استخدام هذا المجلد لتخزين الملفات العامة التي يمكن الوصول إليها من قبل المتصفح.
- يتم استخدام هذا المجلد لتخزين صورًا ورسوماتًا تستخدم في واجهة المستخدم.
- يتم استخدام هذا المجلد لتخزين خطوطًا تستخدم في واجهة المستخدم.
- يتم استخدام هذا المجلد لتخزين ملفات JavaScript العامة التي تستخدم في عرض ملفات PDF.
