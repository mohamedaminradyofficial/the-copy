# دليل المجلد: utils

## وصف المجلد
المجلد `utils` يحتوي على وظائف مساعدة (utilities) تستخدم في جميع أنحاء المشروع. هذه الوظائف توفر وظائف مشتركة ومتكررة لتجنب التكرار في الكود.

## قائمة الملفات ووظائفها

### 1. logger.js
- **الوصف**: نظام تسجيل (logging) متقدم.
- **الوظائف الرئيسية**:
  - تسجيل رسائل الخطأ (error)
  - تسجيل رسائل التحذير (warn)
  - تسجيل رسائل المعلومات (info)
  - تسجيل رسائل التصحيح (debug)
  - ضبط مستوى التسجيل

### 2. saveText.js
- **الوصف**: وظيفة لحفظ محتوى نصي إلى ملف.
- **الوظائف الرئيسية**:
  - حفظ المحتوى النصي إلى ملف
  - إنشاء المجلدات المطلوبة تلقائيًا
  - التحقق من صحة المسار قبل الحفظ

### 3. text-protocol.js
- **الوصف**: وظائف لمعالجة البروتوكول النصي.
- **الوظائف الرئيسية**:
  - استخراج أقسام محددة من النص
  - معالجة النصوص مع مراعاة الأمان

### 4. text-utils.js
- **الوصف**: وظائف مساعدة لمعالجة النصوص.
- **الوظائف الرئيسية**:
  - تحويل المحتوى إلى نص
  - استخراج جزء آمن من النص

## ملاحظات خاصة
- جميع الوظائف مكتوبة باستخدام JavaScript/TypeScript.
- الوظائف مصممة لتكون مستقلة ومتكررة الاستخدام.
- يتم استخدام هذه الوظائف في جميع أنحاء المشروع لتوفير وظائف مشتركة.

## الاعتماديات
- لا يوجد اعتمادات خارجية لهذا المجلد.
- يتم استخدام Node.js API الأساسي مثل `fs` و `path`.

## العلاقات مع مجلدات أخرى
- هذا المجلد يستخدم في جميع المجلدات الأخرى في المشروع.
- يتم استيراد الوظائف من هذا المجلد في جميع أنحاء المشروع.
