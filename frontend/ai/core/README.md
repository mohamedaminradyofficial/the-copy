# core

This directory contains the core AI components and utilities for the narrative analysis system.

## Subdirectories
- `models`: Contains the core data models and entities
- `pipeline`: Contains the pipeline infrastructure components

## Files
- `gemini-core.js`: Core utilities for interacting with the Gemini AI models

## Key Features
- Provides core data models and entities for the analysis system
- Implements the pipeline infrastructure for executing the analysis stations
- Offers core utilities for interacting with the Gemini AI models

## Usage
These components are used internally by the analysis pipeline and are not typically called directly by end users.

## Notes
- The core models define the fundamental data structures used throughout the system
- The pipeline infrastructure provides the framework for executing the analysis stations
- The Gemini core utilities provide essential functionality for interacting with the AI models
