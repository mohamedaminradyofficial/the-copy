# المجلد: /frontend/ai/stations

## الوصف
يحتوي هذا المجلد على محطات التحليل الرئيسية للنظام، بما في ذلك:

- محطات التحليل الأساسية
- محطات التحليل المتقدمة
- محطات التشخيص والتحسين
- محطات التجميع النهائي

## قائمة الملفات ووظائفها

1. `base-station.js`:
   - نظام أساسي قابل للتوسيع لمحطات خط الإنتاج
   - يوفر بنية أساسية لتشغيل مهام التحليل المختلفة
   - يتضمن دعمًا لفحص الإدخال/الخروج وتتبع الأداء
   - يقوم بإدارة الأخطاء والاستجابات الاحتياطية
   - يستخدم سجلًا لتتبع تفاصيل التنفيذ

2. `gemini-service.js`:
   - إعادة تصدير خدمة Gemini من المجلد stations
   - يوفر التوافق مع الإصدارات السابقة من خلال إعادة التصدير
   - مخصص لتوفير موقع مركزي للوصول إلى خدمة Gemini

3. `network-diagnostics.js`:
   - تحليل شبكات الصراع وتحديد المشكلات
   - تشخيص المشكلات الهيكلية والشخصيات المعزولة والصراعات المتروكة
   - تقييم درجة صحة الشبكة وتحديد نقاط الضعف
   - توليد تقارير تشخيصية شاملة

4. `orchestrator.js`:
   - منسق تنفيذ خط الإنتاج السبعة
   - إدارة تدفق البيانات بين المحطات
   - إدارة الأخطاء وإعادة المحاولة
   - تتبع التقدم وحالة التنفيذ
   - إدارة التنفيذ الشامل للنظام

5. `run-all-stations.js`:
   - تشغيل خط الإنتاج الشامل
   - إنشاء وإدارة خطوط الإنتاج
   - تنفيذ المحطات السبعة في تسلسل
   - تقديم مقاييس الأداء ونتيجة التحليل
   - التحقق من صحة النظام والتأكد من جاهزيته

6. `station1-text-analysis.js`:
   - تحليل النص لتحديد الشخصيات الرئيسية وعلاقاتهم
   - توليد خط ملخصي وتحليل النبرة السردية
   - تحليل الشخصيات في العمق، الحوار، والأصوات
   - حساب إحصائيات النص وتقرير عدم اليقين

7. `station2-conceptual-analysis.js`:
   - تحليل الجوانب المفاهيمية للنص
   - توليد بيانات القصة، خرائط ثلاثية الأبعاد، وعروض مختصرة
   - تحديد المواضيع والأنواع، وتحليل الجمهور المستهدف
   - تحليل السوق والتوصيات

8. `station3-network-builder.js`:
   - بناء شبكة الصراع من النص المحلل
   - استنتاج العلاقات والصراعات
   - تحليل الشبكة وتوليد تقارير التشخيص
   - توليد أقواس الشخصيات ونقاط التحول

9. `station4-efficiency-metrics.js`:
   - حساب مقاييس الكفاءة للنص
   - تقييم الجودة الأدبية، إمكانية الإنتاج، والإيقاع
   - توليد توصيات لتحسين النص
   - تقييم الجودة والتحليل السوقي

10. `station5-dynamic-symbolic-stylistic.js`:
    - تحليل الجوانب الديناميكية والرمزية والأسلوبية للنص
    - بناء خط زمني للأحداث وتحليل تطور الشبكة
    - تحليل الرموز والأسلوب، التوتر، والحوار
    - تحليل البصرية والسينمائية

11. `station6-diagnostics-treatment.js`:
    - تشخيص شامل وتحليل العلاج
    - توليد تقارير التشخيص، نقاش متعدد الوكلاء، وتخطيط العلاج
    - توقعات مسار القصة وتقييم عدم اليقين
    - توليد خطط علاجية مفصلة والتوصيات

12. `station7-finalization.js`:
    - تجميع النهائي وتحليل الشامل
    - حساب مصفوفة النتائج، توليد ملخص تنفيذي، وتقييم شامل
    - تحليل SWOT، صدى الجمهور، واقتراحات إعادة الكتابة
    - توليد تقارير قابلة للقراءة بالبشرية وMarkdown

## ملاحظات
- جميع المحطات مصممة لتوفير تحليلات متقدمة للنصوص الدرامية
- المحطات مخصصة لتوسيعها في الأنظمة الفرعية مع تحسينات محددة
- المكونات مخصصة لتوسيعها في المحطات الفرعية مع تحسينات محددة
