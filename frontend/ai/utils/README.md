# utils

هذا المجلد يحتوي على الأدوات المساعدة في نظام التحليل الدرامي.

## الملفات

### الأدوات المساعدة
- `logger.js`: يوفر أداة التسجيل
- `saveText.js`: يوفر أداة حفظ النص
- `text-utils.js`: يوفر أدوات مساعدة للنص

## الميزات الرئيسية
- يوفر أداة التسجيل
- يوفر أداة حفظ النص
- يوفر أدوات مساعدة للنص

## الاستخدام
هذه الأدوات تُستخدم داخليًا من قبل نظام التحليل وتكون غير مرئية للمستخدمين النهائيين.

## ملاحظات
- أداة التسجيل توفر التسجيل في نظام التحليل
- أداة حفظ النص توفر حفظ النص في نظام التحليل
- أدوات مساعدة للنص توفر الأدوات المساعدة للنص في نظام التحليل
