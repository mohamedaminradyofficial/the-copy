# interfaces

هذا المجلد يحتوي على واجهات النظام الأساسية في نظام التحليل الدرامي.

## الملفات

### واجهات النظام
- `gemini-service-boundary.js`: يحدد حدود خدمة Gemini
- `response-types.js`: يحدد أنواع الاستجابات المختلفة
- `stations.js`: يحدد واجهات المحطات المختلفة

## الميزات الرئيسية
- يحدد حدود خدمة Gemini
- يحدد أنواع الاستجابات المختلفة
- يحدد واجهات المحطات المختلفة

## الاستخدام
هذه الواجهات تُستخدم داخليًا من قبل نظام التحليل وتكون غير مرئية للمستخدمين النهائيين.

## ملاحظات
- حدود خدمة Gemini تحدد كيفية تفاعل النظام مع خدمة Gemini
- أنواع الاستجابات تحدد كيفية تنسيق الاستجابات المختلفة
- واجهات المحطات تحدد كيفية تفاعل المحطات المختلفة مع بعضها البعض
